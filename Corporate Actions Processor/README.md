# Corporate Actions Automation Engine

## Overview
Automates processing of corporate actions including dividends, splits, and mergers with position adjustments.

## Workflow Description
Monitors corporate action announcements, processes position adjustments, handles cash distributions, and updates portfolio records automatically.

## Implementation Guide

### 1. Prerequisites
- Corporate actions data
- Portfolio positions
- Adjustment calculations
- Operations notifications

### 2. Database Setup
Create `corporate_actions` and `portfolio_positions` tables.

### 3. n8n Configuration Steps
- Import workflow and configure data sources
- Set up adjustment logic
- Configure notifications

### 4. Testing
Test with sample corporate actions and verify position adjustments.

### 5. Monitoring
Monitor processing success and adjustment accuracy.

### 6. Troubleshooting
Check for data synchronization and calculation errors.
