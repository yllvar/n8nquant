# Portfolio Performance Attribution Engine

## Overview
Performs daily performance attribution using Brinson-Fachler models and factor analysis with contribution analysis.

## Workflow Description
Calculates performance attribution by factors, sectors, and other dimensions, generates analysis, and distributes reports to portfolio managers.

## Implementation Guide

### 1. Prerequisites
- Portfolio data
- Factor models
- Attribution calculation
- Performance reports

### 2. Database Setup
Create `performance_attribution` and `portfolio_positions` tables.

### 3. n8n Configuration Steps
- Import workflow and configure data sources
- Set up attribution calculations
- Configure reporting

### 4. Testing
Test with sample portfolios and verify attribution calculations.

### 5. Monitoring
Monitor attribution accuracy and factor exposure.

### 6. Troubleshooting
Check for data quality and calculation errors.
